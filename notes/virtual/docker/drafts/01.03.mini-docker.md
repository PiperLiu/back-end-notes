### 感受 namespace

```bash
root@piperliu:~# ps
    PID TTY          TIME CMD
  10061 pts/2    00:00:00 bash
 263642 pts/2    00:00:00 bash
 263952 pts/2    00:00:00 ps
root@piperliu:~# docker run --rm -it busybox:latest
/ # hostname
ff3592c1d29e
/ # ps
PID   USER     TIME  COMMAND
    1 root      0:00 sh
    9 root      0:00 ps
/ #
```

如上，在 docker 容器中，各个进程的进程号就很小。因为其进程使用了独有的 `namespace` 。
